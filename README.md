# Locus map

shell script for android terminal - download map [Locus](http://www.locusmap.eu)
> source : <http://osm.paws.cz>

`sh map [MAP]`
  - czech
  - slovakia
  - sachsen
  - bayern
  - austria
  - croatia
  - hungary
  - georgia
  - malta
  - poland

`sh map list`
view download maps
